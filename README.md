# 🛒 E-commerce Frontend (Amazon Clone)

🚀 A responsive frontend e-commerce application inspired by Amazon, featuring user authentication and cart management.

---

## 🚀 Features

* 🔐 User Authentication (Login & Signup UI)
* 🛒 Add to Cart & Remove from Cart
* 📦 Product Listing Page
* 🔍 Basic Search / Filtering (if added)
* 📱 Responsive Design (Mobile + Desktop)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript / React (update accordingly)
* **State Management:** (useState / Context API if used)

The goal of this project was to understand and implement core e-commerce functionalities such as product display, cart management, and user authentication on the frontend.
